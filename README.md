# Клиент-серверное приложение

## 🖥️ Сторона сервера

Перед запуском необходимо узнать IP-адрес машины и сообщить его клиенту.

**Шаг 1 — Узнать IP-адрес**

Откройте командную строку (`cmd`) и выполните команду:

```
ipconfig
```

В выводе найдите строку **IPv4-адрес** (например: `192.168.1.100`) и передайте его клиенту.

**Шаг 2 — Запустить сервер**

Запустите серверную программу и ожидайте подключения клиента.

---

## 💻 Сторона клиента

**Шаг 1 — Получить IP-адрес сервера**

Уточните у администратора сервера его IP-адрес до запуска программы.

**Шаг 2 — Запустить программу**

При запуске программы введите IP-адрес сервера:

```
Введите IP-адрес сервера: 192.168.1.100
```

После успешного подключения программа перейдёт в рабочий режим.

---

## Краткая сводка

| Сторона   | Действие |
|-----------|----------|
| **Сервер** | Запустить `ipconfig`, узнать IPv4-адрес, сообщить клиенту, запустить сервер |
| **Клиент** | Получить IP-адрес от сервера, запустить программу, ввести адрес при старте |
